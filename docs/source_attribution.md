# 專案來源說明

## 原始系統

本專案基於以下開源系統重新實現：

- **系統名稱**: MULTALL Turbomachinery Design System
- **來源網站**: https://sites.google.com/view/multall-turbomachinery-design/to-download-the-system
- **原始版本**: multall-open_20260120
- **原始語言**: FORTRAN 77
- **原始檔案**: 位於 `multall-open_20260120/` 目錄（本地參考用，不包含在 Git 倉庫中）

## 專案目標

將原始的 FORTRAN 77 程式碼現代化，使用 Python 3.14 和 PySide6 重新實現，提供：

1. 現代化的圖形使用者介面
2. 更好的跨平台支援
3. 完整的正體中文/UTF-8 支援
4. 更易於維護和擴展的程式碼結構
